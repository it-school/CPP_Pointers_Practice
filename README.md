# Примеры использования указателей и динамической памяти в C++

## Описание проекта

Этот проект содержит набор примеров для учебной лекции по теме указателей и динамической памяти в C++.
Он демонстрирует различные аспекты работы с указателями, динамическим выделением памяти, передачей данных по адресу и диагностикой утечек памяти.

Проект ориентирован на студентов и разработан для пошагового изучения с комментариями прямо в коде.

## Содержание проекта

Примеры организованы по возрастанию сложности:

1. **Передача переменной по адресу** – демонстрация изменения значения переменной через указатель.
2. **Динамическое выделение одного числа** – использование `new` с проверкой успешности выделения.
3. **Изменение размера массива** – создание нового массива большего размера и перенос данных.
4. **Передача массива в функцию** – заполнение массива значениями через указатель.
5. **Указатель на указатель** – динамическое выделение памяти через двойной указатель.
6. **Передача нескольких переменных по указателю** – изменение нескольких значений через функцию.
7. **Демонстрация утечки памяти с мониторингом** – наблюдение состояния доступной памяти до и после выделения.
8. **Разница между malloc/free и new/delete** – сравнение C-style и C++-style управления памятью.

## Особенности

* Расширенные комментарии для понимания механизма работы указателей.
* Встроенные проверки успешности выделения памяти.
* Демонстрация как безопасного, так и опасного (утечка памяти) использования.
* Примеры работают на Windows и показывают базовую диагностику состояния памяти.

## Требования

* Компилятор C++11 и выше.
* ОС Windows для корректного отображения информации о памяти (GlobalMemoryStatusEx). Для других ОС вывод памяти будет недоступен.

## Сборка и запуск

1. Скопируйте код проекта в файл `main.cpp`.
2. Скомпилируйте его с помощью любого современного компилятора C++:

   ```bash
   g++ main.cpp -o pointers_example.exe
   ```
3. Запустите программу:

   ```bash
   ./pointers_example.exe
   ```

Программа пошагово выполняет все примеры и выводит результат на консоль.

## Важные замечания

* Пример 7 специально содержит утечку памяти для учебных целей. **Не повторяйте такие практики в реальных проектах**.
* Использование `malloc/free` и `new/delete` показано для сравнения стилей. В современном C++ предпочтителен `new/delete` или ещё лучше умные указатели (`std::unique_ptr`, `std::shared_ptr`).
